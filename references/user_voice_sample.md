# 我的声音样本（用于 AI 模仿）

这是我未经整理的口述，最接近我真实的思考和表达方式。AI 在写作时，应该模仿这种“想到哪说到哪”的语感，而不是追求工整和书面化。

```
啊，就是我最近在用 Kimi cloud，然后这个是我开了一个 Kimi 199 的会员之后，有了一个功能。然后这个其实就是把 open cloud 在云端部署，然后通过 Kimi 去做交互，Kimi 去做一个基座模型。

而我这几天一直在想，我该怎么让它成为跟我别的大模型、跟我别的一个 AI 助手不一样的一个存在？它到底通过这种独属于我服务器的情况下，能够向我提供别的服务器提供不了什么样的价值啊？

我一直在思考这个事儿，然后我也一直在想各种各样的途径，包括目前在想，他能不能我给他搞一个 boss 直聘的小号，然后他替我去筛选一些岗位，然后我再用我的主号去进行投递，等等等等。
```
