【全局设计风格锁｜所有选题通用】

目的：让你后续任何新内容的配图都保持同一套“系列感”（统一画风 + 主角一致性），不需要每次重新沟通画风。

使用方式（强制）：
- 生图时必须把本文件作为提示词前缀（prompt prefix）。
- 只允许使用调用方提供的真人参考图作为唯一参考（禁止其它参考图）。
- 角色必须为男性，并且必须以该真人参考图为原型（每次生成都要带参考图）。

通用硬规则（不要改动，除非你明确要改风格）：
- 风格：清新、干净、克制的 Q 版信息图插画（不是写实、不是3D、不是“科技海报”）
- 线条：粗黑描边（线粗统一），形状圆润
- 上色：扁平上色 + 极轻的柔和阴影（不要厚重阴影/高光/金属质感/体积光）
- 背景：柔和渐变 + 极轻纸感纹理；留白充足
- 构图：画面干净、元素少、信息密度低（不要堆图标/不要复杂场景堆叠）

文字规则（强制｜用于避免“9:16版”之类的脏字）：
- 如果调用方提供了「On-image text（标题/要点/标签）」清单：只允许出现清单里的文字，禁止模型自行发明任何额外文案。
- 如果调用方明确要求“无文字/封面无字”：整张图必须 0 文字（包括中文/英文/数字/符号/水印/角标）。
- 禁止任何“版式/比例/版本”角标或说明：例如“9:16版 / 竖图版 / 横图版 / 比例:… / 移动端全屏报 / 适配手机”等字样。

禁止项（强制）：
- 不要水印/Logo/二维码/角标/版本标记
- 不要 UI 截图、不要 PPT 模板、不要真实软件界面
- 不要强烈 3D 体积光/金属质感/厚重高光（避免变成“3D 海报”）

角色一致性（只允许真人图参考）：
- 每次都参考 portraits 真人照：尽量还原发型轮廓/眉毛浓度/脸型气质（但保持 Q 版简化：点眼睛/腮红/简化五官）
- 必须：干净无胡子、无胡茬
- 发型必须：自然侧分短发 + 少量刘海自然垂落；头发贴头皮、哑光、不油亮、低体积不蓬松
- 严禁发型：油头/背头/大背头/飞机头/纹理烫/卷发/寸头/渐变推短/undercut/莫西干/染发
- 服饰（系列统一）：深色连帽卫衣
- 性别：男性
