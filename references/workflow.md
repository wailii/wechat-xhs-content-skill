# WeChat Structure Templates

## Default Narrative

1. 个人或行业痛点开场
2. 具体问题拆解（3–5条）
3. 方法/流程拆解（步骤化）
4. 结果对比（时间/效率/质量）
5. 方法论升华 + 互动提问

## Variant A: 清单干货

- 直接给结论
- 3–7 个可执行要点
- 每点给一个小例子
- 最后补一个“适用场景/不适用场景”

## Variant B: 观点短论

- 先抛观点
- 用 1–2 个真实案例支撑
- 给出可操作的下一步

## Section Length Guidance

- 引子 10–15%
- 问题与方法 60–70%
- 结果与升华 20–25%
