# Xiaohongshu Visual & Storyboard Guide

## Page Count

- Let content determine image count; do not force a fixed number.
- Required: cover + 3–6 content pages + closing CTA

## Layout Rules

- 每页 1 个大标题 + 2–4 条要点
- 关键数字或对比至少出现 2 页
- 留白充足，避免信息堆叠

## Visual Style

- Q 版人物形象统一
- 轻量卡通、柔和配色、清晰图标
- 背景可变化：暖色渐变/柔和浅色块

## Typical Page Types

1. Cover: 大标题 + 人物 + 强对比数字
2. Why: 痛点与场景
3. Before/After: 效率或流程对比
4. Workflow: 流程图或步骤条
5. Tools/Setup: 工具与方法
6. Results: 数据或结果清单
7. Reflection/CTA: 观点升华 + 互动提问

## Caption Pattern

- 3–5 段短文案
- 中间穿插 1–2 个小标题
- 结尾提问 + 5–8 个标签
